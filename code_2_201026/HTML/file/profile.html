关于王一博

2014

2014年9月15日，以UNIQ组合成员身份正式出道。

2015

2015年，主演青春励志电影《梦想合伙人》。
2015年4月29日，正式成为综艺节目 《天天向上》主持团“天天兄弟”成员； 12月，领衔主演都市爱情励志喜剧 《人间至味是清欢》。

2017

2017年，主演青春校园仙侠剧 《私立蜀山学园》； 同年，获得第十七届音乐风云榜年度盛典偶像新势力奖、亚洲影响力盛典最佳新人演员奖以及微博电视影响力盛典年度新锐艺人奖。

2018

2018年2月，主持湖南卫视春节联欢晚会；3月，担任竞演类真人秀节目《创造101》舞蹈导师； 4月，领衔主演仙侠古装剧《陈情令》。 9月，主演青春电竞电视连续剧《陪你到世界之巅》。

2019

2019年9月，主演古装剧《有翡》。10月，王一博入选2019福布斯中国30位30岁以下精英榜。

2020

2020年5月，主演刑侦涉案剧《冰雨火》。10月18日，凭借电视剧《陪你到世界之巅》获得第30届中国电视金鹰奖观众喜爱的男演员奖。
